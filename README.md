# 收藏+（jdsc）

一个用于 ComfyUI 的“节点收藏与搜索”前端扩展与后端节点，支持快速收藏单节点、收藏工作流片段，两列收藏展示、智能搜索（含中文）、可拖动悬浮图标与组合快捷键。

## 安装
- 将本项目目录 `jdsc` 放入 `ComfyUI/custom_nodes/` 下（例如 `O:\ComfyI2I\ComfyUI\custom_nodes\jdsc`）
- 启动 ComfyUI，前端会自动加载 `web/jdsc.js`
- 节点会以展示名 `收藏+` 出现在分类 `wuhuo` 下，用于提供前端面板资源

## 节点
- 节点内部名：`jdsc`
- 展示名：`收藏+`
- 分类：`wuhuo`
- 用于提供 `WEB_DIRECTORY`，让前端扩展自动注入

## 前端功能
- 悬浮图标
  - 右下角心形悬浮按钮，可拖动
  - 位置记忆：拖动后记录到 `localStorage`，下次自动恢复
- 组合快捷键
  - 支持 `Ctrl/Alt/Shift/Meta` 与任意键组合，例如 `Ctrl+Alt+F`、`Shift+Enter`
  - 在面板右下点击“设置快捷键”修改，保存在 `jdsc:hotkey`
- 面板结构
  - 顶部标签：`搜索`、`收藏`（等宽）
  - 收藏页二级标签：`节点收藏`（橙色）、`片段收藏`（紫色）
  - 输入框带清空按钮：单击 `×` 清除当前关键字
  - 语言切换：右上角中英切换图标（保存在 `jdsc:lang`）

## 搜索（与系统一致的匹配增强）
- 索引字段：名称、类型路径、描述、项目/分支路径
- 英文容错：`l/i` 视觉相近字符归一化
- 中文增强：去分隔符匹配（空格/下划线/斜杠），中文与英文混合词分词 AND 命中
- 忽略符号：slug 匹配（仅英文关键字时）
- 性能优化：索引预热、输入防抖、分块渲染（首批 60/120 条，滚动追加）

## 收藏
- 数据存储（`localStorage`）
  - `jdsc:favorites`：单节点收藏（含自定义名与原名）
  - `jdsc:groups`：分组收藏（选中节点集合，可含连线）
  - `jdsc:frags`：工作流片段（位置、参数与内部连线）
- 收藏页展示（两列）
  - 默认：左列“节点收藏”、右列“片段收藏”（不重复）
  - 切到“节点收藏/片段收藏”：两列分别只显示对应类型（交替分配）
- 三行信息（节点）
  - 行1：自定义名称（右键改名）
  - 行2：`❤` 项目/分支路径（如 `IhyNode/WanAnimate`）
  - 行3：原节点名（小字）
- 右键改名
  - 节点/分组/片段均支持右键改名；取消双击改名以避免误触
- 取消收藏
  - 点击心形图标弹确认；仅切换收藏状态（阻止冒泡），不会触发添加到画布
- 过滤
  - 收藏页与搜索页共用同一过滤逻辑；切页后输入框保留的关键词会立即过滤收藏列表

## 使用示例
- 打开面板：点击右下角心形悬浮按钮，或使用设置好的组合快捷键
- 收藏单节点：在搜索页输入关键字，点击结果右侧心形 `♡`
- 收藏片段：框选画布中的多个节点，点击面板底部“收藏片段”按钮
- 收藏分组：选择若干节点，点击“收藏节点”保存为分组
- 添加到画布：在收藏或搜索结果中，单击条目名称即可添加，自动避让并居中显示，保持选中
- 取消收藏：点击心形按钮切换为 `♡`，条目不会被添加到画布
- 语言切换：点击标题右侧图标，中英文即时切换

## 添加到画布
- 单节点
  - 避让：全局搜索不相交空白矩形（含复查与搬移），避免覆盖现有节点
  - 居中：添加后视图自动居中新节点
  - 选中：保持新节点为选中状态
- 片段
  - 恢复：位置与内部连线；在空白区域整体放置，矩形居中
  - 分组框：自动创建并命名为片段收藏名，框住片段范围
- 分组（节点集合）
  - 按网格估算尺寸放置在空白区；无显式连线时按端口类型尝试自动连线

## UI 细节
- 心形收藏按钮紧凑（18×18），支持即时变色
- 搜索与收藏切换时清除搜索页滚动处理器，避免未收藏内容混入收藏页
- 面板与图标位置均支持记忆：`jdsc:modalPos`、`jdsc:floatPos`

## 兼容性与依赖
- 依赖 ComfyUI 前端环境与 LiteGraph（可选 `LGraphGroup`）
- 后端仅使用 `os` 标准库；无第三方依赖

## 常见问题
- 收藏页出现未收藏内容：已在切换到收藏时撤销搜索滚动处理器；刷新前端如仍存在，可清空 `jdsc:favorites/jdsc:groups/jdsc:frags`
- 中文搜索差异：已增强为与系统一致的匹配管线；若个别词条仍异常，请提供词条与目标节点名以便微调
