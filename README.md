# whtools (原 jdsc)

一个用于 ComfyUI 的综合工具箱，包含“节点收藏”、“工作流管理”以及多种实用的辅助节点。

## 安装

1.  将本项目目录 `jdsc` 放入 `ComfyUI/custom_nodes/` 下 (例如 `ComfyUI\custom_nodes\jdsc`)
2.  启动 ComfyUI，前端会自动加载相关扩展。
3.  节点会以展示名 **whtools** 或具体功能名出现在分类 `wuhuo` 下。。

## 核心功能

*   **节点收藏**: 快速收藏常用节点，支持拖拽生成的悬浮球进行管理。
*   **工作流片段收藏**: 将选中的多个节点保存为片段，方便复用。
*   **智能搜索**: 支持中文拼音搜索节点。
*   **工作流管理**: 提供工作流文件的分类管理功能。

## 节点介绍

### 1. 文本传递 (WuhuoTextGate)
一个强大的文本处理节点，支持“红绿灯”逻辑，可用于控制工作流的执行。

*   **功能**:
    *   **直通模式 (Free Pass)**: 绿灯。如果有上游输入，则传递上游内容；如果没有，则使用节点内的编辑内容。
    *   **编辑模式 (Enable Edit)**: 红灯。暂停工作流，允许用户在节点内编辑文本。
    *   **强化预设**: 内置多种提示词预设（如画质增强等），可叠加到文本前。
    *   **关键词前缀**: 可设置固定的关键词前缀。

### 2. 多图预览 (WuhuoMultiPreview)
支持多组图片预览与历史记录管理的节点。

*   **功能**:
    *   **多组输入**: 支持 `images` 和 `images_opt` 两组输入，方便对比。
    *   **历史记录**: 自动保存生成的图片历史，支持在节点上直接回看。
    *   **右键菜单**: 提供“打开保存目录”等便捷操作。
    *   **动态布局**: 根据图片数量和节点大小自动调整网格布局。

### 3. 空latent+ (WuhuoEmptyLatentVideo)
专为视频生成设计的 Empty Latent 节点。

*   **功能**:
    *   **时长换算**: 直接输入视频时长（秒），自动换算为所需的帧数 (frames)。
    *   **比例预设**: 提供 16:9, 9:16, 4:3 等常用视频比例。
    *   **横竖屏切换**: 一键切换横屏/竖屏。
    *   **自动对齐**: 宽高自动对齐到 8 的倍数，避免报错。

### 4. 空latent+Qwen (WuhuoEmptyLatentQwen)
专为文生图 (如 Flux, Qwen) 设计的 Empty Latent 节点。

*   **功能**:
    *   **Batch Size**: 支持设置生成数量 (Batch Size)。
    *   **比例预设**: 提供常用图片比例。
    *   **大分辨率支持**: 默认参数适配高质量生成需求。

### 5. 忽略选择框 (WuhuoIgnoreGroup)
配合前端功能的辅助节点。

*   **功能**:
    *   用于标记一组节点是否被“忽略”或“Bypass”。

### 6. 工作流+ (WuhuoWorkflowManager)
工作流管理器的后端支持节点。

*   **功能**:
    *   提供工作流文件的读取、保存、分类管理等 API 支持。

## 使用说明

*   **悬浮球**: 页面右下角会出现“收藏+”悬浮球，点击可展开收藏面板。
*   **快捷键**: 支持配置快捷键呼出搜索面板。
